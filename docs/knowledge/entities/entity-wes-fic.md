# Wes-FIC (Wesco Financial Insurance Company)

---

## 概述

Wes-FIC 是 Wesco Financial Corporation 的保险子公司，在Wesco 1997年致股东信中被芒格多次提及。它主要经营超级巨灾再保险业务，是Wesco保险业务板块的重要组成部分。

## 基本信息

- **类型**: Company
- **行业**: 保险/再保险
- **母公司**: Wesco Financial Corporation
- **最终母公司**: Berkshire Hathaway

## 关键信息

- Wes-FIC 在超级巨灾再保险领域运营
- 该业务需要"堡垒级资产负债表"作为支撑
- 年度结果波动性极大，"某些非常不愉快的年份是必然的"
- 业务的风险特征不适合胆小的投资者
