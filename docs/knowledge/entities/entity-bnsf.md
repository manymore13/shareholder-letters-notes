# BNSF 铁路

---

## 概述

BNSF铁路（Burlington Northern Santa Fe Railway）是美国最大的货运铁路公司之一，成立于1849年。2010年被伯克希尔以440亿美元收购，是伯克希尔历史上最大的收购之一。

**类型**：组织 / 全资子公司  
**行业**：铁路运输  
**母公司**：[entity-berkshire-hathaway](./entity-berkshire-hathaway.md)
