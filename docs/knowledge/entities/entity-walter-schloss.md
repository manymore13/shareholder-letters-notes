# Walter Schloss

## Overview

Walter Schloss（1916-2012）是传奇价值投资者，本杰明·格雷厄姆的学生之一。他没有上过大学，仅通过格雷厄姆的夜校课程学习投资。他在长达28年的投资生涯中创造了卓越的长期回报，以极度分散的策略著称，持有多达100只以上股票，只关注价格与价值之间的差距，不看行业本质。

## Background

- **Type**: Person
- **Born**: 1916年
- **Died**: 2012年
- **Education**: 格雷厄姆夜校课程（未上大学）
- **Career**: 1955年创立自己的投资合伙公司，管理至2000年代

## Key Information

### 投资风格

- **极度分散**：持有100只以上股票，远超大多数价值投资者
- **简单方法**：不看行业本质，只看价格与账面价值之间的差距
- **独立思考**：与格雷厄姆-多德村落的其他投资者持仓几乎不重叠
- **低成本运作**：办公室简朴，不使用分析师团队

### 投资业绩

- 1955-2002年：28年复合年化回报约15.3%（同期标普约10%）
- 被巴菲特在1984年哥伦比亚大学演讲中列为"超级投资者"之一
