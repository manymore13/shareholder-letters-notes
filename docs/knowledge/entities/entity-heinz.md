# 亨氏

---

## 概述

亨氏（The H.J. Heinz Company）是全球著名的食品公司，以番茄酱闻名。2013年与伯克希尔及3G资本联合以230亿美元收购，后与卡夫合并成为卡夫亨氏（Kraft Heinz）。

**类型**：组织 / 上市公司  
**行业**：食品制造  
**持股关系**：[entity-berkshire-hathaway](./entity-berkshire-hathaway.md)

---

## 关键信息

- **成立时间**：1869年
- **总部**：美国宾夕法尼亚州匹兹堡
- **伯克希尔持股**：通过合资企业

## 投资历程

- 2013年2月完成收购
- 收购价格：230亿美元
- 伯克希尔提供融资和投资
