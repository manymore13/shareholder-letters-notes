# 内在价值估算

---

## 概述

内在价值估算（Intrinsic Value Estimation）是对企业真实经济价值进行定量评估的过程。芒格在Wesco 1997年致股东信中提出了一种独特的估算方法，结合了清算价值和递延税优势两个因素。

**类型**：方法论
**类别**：投资分析
**领域**：价值投资

---

## 核心定义

### 基本概念

内在价值估算旨在确定企业的"真实"经济价值，为投资决策提供量化依据。与[内在价值](concept-intrinsic-value.md)这一理论概念不同，内在价值估算关注的是具体的计算方法。

### 芒格的方法

芒格在Wesco 1997年信中提出的估算方法包括两个核心要素：

1. **清算价值**：如果企业被清算，资产变现后股东能获得的实际金额
2. **递延税优势**：企业因持有未实现增值资产而获得的无息"贷款"

---

## 估算方法的要素

### 清算价值评估

- 资产的可变现净值（而非账面价值）
- 扣除清算成本和负债
- 考虑资产的实际市场流动性

### 递延税优势

- 政府因递延税款而提供的无息"贷款"
- 资产持有时间越长，优势越大
- 需要区分税务负债和税务优势
