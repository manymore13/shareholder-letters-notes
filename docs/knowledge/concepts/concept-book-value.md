# 账面价值

---

## 概述

账面价值（Book Value）是企业会计上的净资产值，等于总资产减去总负债。对于伯克希尔来说，账面价值是衡量公司增长的重要指标，虽然巴菲特强调它远低于公司的内在价值。

**类型**：指标  
**类别**：财务分析  
**领域**：价值投资

---

## 核心定义

### 基本公式

```
账面价值 = 总资产 - 总负债
```

### 每股账面价值

```
每股账面价值 = 账面价值 / 流通股份数
```

---

## 账面价值 vs 内在价值

### 关键区别

| 特征 | 账面价值 | 内在价值 |
|------|---------|---------|
| **定义** | 会计净资产 | 未来现金流折现 |
| **依据** | 历史成本 | 未来预期 |
| **波动** | 相对稳定 | 持续变化 |
| **与股价关系** | 近似但不等于 | 决定长期股价 |

### 巴菲特的观点

> "账面价值是内在价值的粗略替代，但通常低估了我们的真实价值。"

伯克希尔的内在价值远超账面价值，原因：
- 保险浮存金的价值
- 经济商誉
- 留存收益的再投资

## 历史数据

### 伯克希尔的增长

| 时期 | 账面价值增长 |
|------|------------|
| 1965-2023 | 年化约19.8% |
| 期间倍数 | 超过4万倍 |

## 备注

账面价值是衡量企业规模的有用指标，但不是衡量内在价值的最佳方式。投资者应该关注企业创造价值的能力，而非单纯的账面数字。
