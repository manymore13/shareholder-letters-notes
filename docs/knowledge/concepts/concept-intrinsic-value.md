# 内在价值

---

## 概述

内在价值（Intrinsic Value）是巴菲特投资哲学的核心概念，指的是企业未来现金流量的折现值。它代表了企业的真实经济价值，是投资决策的基础。巴菲特强调：价格是你付出的，价值是你得到的。

**类型**：理论  
**类别**：投资分析  
**领域**：价值投资

---

## 核心定义

### 基本概念

内在价值是一个投资标的"真实"的经济价值，基于对未来现金流的合理估计：

> **内在价值 = 企业生命周期内未来自由现金流的折现值之和**

### 与市场价值的关系

- **市场价格**：每天波动的报价
- **内在价值**：基于基本面的估算
- **关系**：长期来看，价格会向价值回归

---

## 计算方法

### 1. 折现现金流法 (DCF)

```
内在价值 = Σ [FCFt / (1+r)^t] + 终值 / (1+r)^n
```

其中：
- FCFt = 第t年的自由现金流
- r = 折现率
- n = 预测期

### 2. 格雷厄姆公式

本杰明·格雷厄姆提出的简化公式：

```
内在价值 = EPS × (8.5 + 2g)
```

其中：
- EPS = 每股收益
- g = 预期增长率

### 3. 清算价值法

对于资产密集型企业：
```
内在价值 = 有形资产 - 负债
```

### 4. 芒格估算法（Wesco 方法）

查理·芒格在 1997 年 Wesco 股东信中提出了一种针对投资组合型公司的实用估算方法：

```
内在价值 = 清算价值 + 递延税优势价值
```

**Wesco 1997 年案例**:
| 组成部分 | 价值/股 |
|---------|--------|
| 清算价值 | ~$248 |
| 递延税优势（名义$102） | ~$25 |
| **内在价值** | **~$273** |
| 年末股价 | $300 |

**递延税优势的逻辑**：
- 未实现资本利得的递延所得税相当于政府的无息"贷款"
- 名义金额（如$102/股）不能全额计入价值
- 实际价值取决于这笔"贷款"能使用多久
- 芒格估计 Wesco 的递延税优势价值约$25/股

---

## 巴菲特的观点

### 估值的艺术

> "内在价值是一个估计值，而不是精确数字。模糊的正确胜过精确的错误。"

巴菲特强调：
- 内在价值是估计值，不是精确数字
- 使用保守的假设
- 关注"模糊的正确"

### 内在价值 vs 账面价值

- **账面价值**：会计记录的历史成本
- **内在价值**：企业未来的盈利能力
- 伯克希尔的内在价值远超账面价值

---

## 投资应用

### 买入时机

- 当市场价格 < 内在价值时买入
- 安全边际越大越好
- 等待"明显的机会"

### 卖出决策

- 当市场价格 >> 内在价值时考虑卖出
- 或找到更好的投资机会
- 优质企业可以永远持有

### 持有策略

对于真正优质的企业：
- 内在价值持续增长
- 市场价格短期可能偏离
- 时间是优秀企业的朋友
