# 查理·芒格

查理·芒格（1924-2023），伯克希尔·哈撒韦副董事长，Wesco Financial 董事长，多元思维模型倡导者。

## 核心思想

芒格以其独特的思维方式和坦诚著称：

- **多元思维模型**：从多个学科角度分析问题
- **逆向思维**："反过来想，总是反过来想"
- [逆向思维](../../knowledge/concepts/concept-inversion.md)方法的应用
