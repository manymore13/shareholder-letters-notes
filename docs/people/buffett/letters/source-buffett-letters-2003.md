# 巴菲特致股东信 2003

## 核心摘要

2003年，伯克希尔的每股账面价值增长21.0%，在过去39年中年复合增长率为22.2%。这一年完成了Clayton Homes的收购，讨论了制造业住房行业的问题以及如何避免投资错误。

## 关键洞察

### 1. Clayton Homes收购

收购背景：
- 田纳西大学学生赠送Jim Clayton自传
- 了解制造业住房行业的"优质企业"
- Kevin Clayton的卓越管理

收购逻辑：
- 行业最佳企业
- 管理层优秀
- 价格合理

### 2. 制造业住房行业问题

行业困境：
- 销售和融资实践糟糕
- 许多不应买房的人买了房
- 许多不应放贷的人放了贷
- 大量止赎和低回收率

Oakwood的教训：
- 投资垃圾债券
- 不理解行业融资问题
- 公司很快破产

Clayton的优势：
- 比竞争对手更负责任
- 财务稳健

### 3. 投资机会的减少

规模影响：
- 股票持仓占净资产比例下降
- 1980年代平均114%
- 2000-2003年平均50%
- 年度表现与标普500关联度降低

挑战：
- 难以找到大量被低估的股票
- 必须部署的资金规模巨大
- 能"推动业绩指针"的股票数量很少

### 4. 经营收益增长

展示长期收益增长：
- 1968-2003年复合增长率：22.8%
- 1973-2003年：20.8%
- 1988-2003年：18.6%
- 2003年经营收益：54.22亿美元

### 5. 资本配置原则

保持灵活：
- 股票便宜时买入股票
- 债券有吸引力时买入债券（如2002年）
- 企业符合标准时收购企业
- 资本闲置痛苦，但比做蠢事好

## 投资理念

### 收购策略

收购Clayton的理由：
- 阅读创始人自传
- 评估CEO能力
- 研究公开财务数据
- 从错误中学习（Oakwood）

### 行业分析

理解行业问题：
- 制造业住房行业的融资问题
- 证券化导致道德风险
- Clayton相对优秀的实践

### 规模挑战

承认现实：
- 股票机会减少
- 必须寻找企业收购
- 资本闲置不可避免

## 数据要点

- 每股账面价值增长：21.0%
- 39年复合增长率：22.2%
- 净资产：约500亿美元
- Clayton Homes收购完成
- 经营收益：54.22亿美元

## 注释

2003年是巴菲特展示如何从错误中学习的一年。Oakwood投资的失败让他更深入理解制造业住房行业，从而识别出行业中的优质企业Clayton Homes。这封信也坦诚讨论了规模对投资机会的影响，承认股票投资占净资产比例下降的现实。
