# 巴菲特致股东信 1992

## 核心摘要

1992年，伯克希尔的每股账面价值增长20.3%，在过去28年中年复合增长率为23.6%。这一年，巴菲特完成了对Lowell鞋业的收购，并再次讨论了内在价值的评估方法和"安全边际"的重要性。

## 关键洞察

### 1. 内在价值评估

深入讨论[内在价值](../../knowledge/concepts/concept-intrinsic-value.md)：

定义：
- 企业未来现金流的折现值
- 不是精确数字，而是估计范围
- 需要合理的保守假设

计算方法：
- 预测未来现金流
- 选择合适的折现率
- 计算现值

对比：
- 账面价值：历史投入
- 内在价值：未来产出
- 市场价值：当前报价

### 2. 安全边际

强调[安全边际](../../knowledge/concepts/concept-margin-of-safety.md)的重要性：
- 以低于内在价值的价格买入
- 容错空间
- 抵御不确定性

原则：
- 不追求精确估值
- 宁可模糊正确，不要精确错误
- 留出安全边际

### 3. Lowell鞋业收购

收购Lowell鞋业：
- 女鞋和护士鞋制造商
- 与H.H. Brown形成协同
- 管理层优秀

收购逻辑：
- 补强现有业务
- 管理层质量
- 合理价格

### 4. 透视盈余

继续讨论[透视盈余](../../knowledge/concepts/concept-look-through-earnings.md)：
- 主要投资对象的盈余
- 未分配利润的价值
- 反映真实经济收益

主要贡献者：
- [Coca-Cola](../../knowledge/entities/entity-coca-cola.md) - 可口可乐
- [GEICO](../../knowledge/entities/entity-geico.md) - 政府雇员保险公司
- [Gillette](../../knowledge/entities/entity-gillette.md) - 吉列

### 5. 市场估值

讨论股市估值水平：
- 1992年市场估值偏高
- 难以找到便宜股票
- 保持耐心

策略：
- 不因现金过多而降低标准
- 宁愿持有现金等待
- 只在机会出现时行动

## 投资理念

### 价值评估原则

内在价值评估：
- 关注未来现金流
- 使用保守假设
- 承认不确定性
- 留出安全边际

### 收购策略

收购标准保持一致：
- 不降低标准
- 不因资金闲置而妥协
- 耐心等待机会

### 风险管理

通过安全边际管理风险：
- 价格保护
- 假设保守
- 承认不确定性

## 注释

1992年，巴菲特详细阐述了内在价值的评估方法和安全边际的重要性。这一年市场估值较高，巴菲特强调保持耐心，不愿降低收购标准。Lowell鞋业的收购补强了伯克希尔的鞋业业务。这封信再次强调了价值投资的核心原则：以低于内在价值的价格买入优质企业。
