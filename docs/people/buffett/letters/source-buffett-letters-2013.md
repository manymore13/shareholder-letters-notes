# 巴菲特致股东信 2013

## 概述

2013年是伯克希尔继续稳健增长的一年。账面价值增长18.2%，完成了两笔重大收购（NV Energy和Heinz），并创造了与3G Capital的合作模式。"五大金刚"（Powerhouse Five）创造了108亿美元的税前收益记录。

## 核心要点

### 重大收购与合作模式

- **NV Energy收购**：以56亿美元收购，供应内华达州88%人口的电力
- **Heinz收购**：与3G Capital合作，创造新型合作模板
  - 伯克希尔作为融资合作伙伴，购买80亿美元优先股（9%股息，预期年回报率12%）
  - 双方各出资42.5亿美元购买普通股
  - **永不出售**：明确表示永不出售任何股份，希望购买更多

### "五大金刚"（Powerhouse Five）

五家最大的非保险业务，合计税前收益108亿美元：
- Berkshire Hathaway Energy（原名MidAmerican Energy）
- BNSF铁路
- IMC（Iscar）
- Lubrizol
- Marmon

**9年增长**：从2004年的3.93亿美元增长到108亿美元，仅伴随6.1%的股份稀释。

### 保险业务持续卓越

- **连续11年承保盈利**，合计220亿美元税前利润
- 浮存金从410亿美元增长到770亿美元
- GEICO成为美国第二大汽车保险公司（超越Allstate）

### "四大金刚"投资

在四家主要投资中增加持股：
- **Wells Fargo**：8.7% → 9.2%
- **IBM**：6.0% → 7.8%
- **Coca-Cola**：8.9% → 9.1%（通过回购）
- **American Express**：13.7% → 14.2%（通过回购）

**重要洞察**：每增加0.1%持股，年度收益增加5000万美元。

### 内在价值vs账面价值

- **内在价值远超账面价值**，差距近年显著扩大
- 股票回购门槛设定在账面价值的120%
- 2013年因股价未降至该水平，未执行回购

### 美国经济乐观主义

引用BNSF收购（2009年）："对美国经济未来的全押赌注"
- 过去237年，押注美国总是正确的选择
- 市场经济的动力将继续发挥魔力
