# 巴菲特致股东信 2006

## 核心摘要

2006年，伯克希尔的每股账面价值增长18.4%，在过去42年中年复合增长率为21.4%。这一年创下了169亿美元的单年净资产增长记录。主要得益于保险业务的幸运之年（无重大飓风）和GEICO生产力的显著提升。

## 关键洞察

### 1. 创记录的年份

成就：
- 净资产增长169亿美元
- 美国企业史上最大单年净资产增长（除合并）
- 42年年复合增长率：21.4%

幸运因素：
- "大自然母亲休假了"
- 2004-2005年飓风损失后，2006年无重大灾害
- 超级巨灾保险从亏损转为巨额盈利

### 2. GEICO的卓越表现

Tony Nicely的成就：
- 45年前18岁加入GEICO
- 1992年成为CEO后增长爆发

生产力提升（2003-2006）：
- 保单数量：570万 → 810万（增长42%）
- 员工数量：减少3.5%
- 生产力提升：47%
- "GEICO起步时就不臃肿"

广告投入：
- 2006年：6.31亿美元
- 2003年：2.38亿美元
- 1995年：3100万美元
- "远超竞争对手，甚至包括更大的公司"

Don Keough的建议：
- "别管生孩子了，让股东立即把现有孩子改名Tony或Antoinette"
- 署名"Tony"

### 3. 大企业的挑战

规模问题：
- 21.7万员工
- 年收入接近1000亿美元

规模导致：
- 思维迟缓
- 抵抗变革
- 自满

历史教训：
- 1965年十大非石油公司（通用汽车、西尔斯、杜邦、柯达等）
- 仅一家进入2006年榜单

丘吉尔名言：
- "我们塑造建筑，之后建筑塑造我们"

### 4. 管理哲学

巴菲特的管理方式：
- "我走了容易的路"
- 坐在后面，让优秀管理者经营
- 任务：加油鼓劲、塑造企业文化、重大资本配置决策

管理者回报：
- 努力工作
- 高效执行

### 5. 两个关键指标

**每股投资**（1965-2006）：
- 1965年：4美元
- 2006年：80,636美元
- 复合增长率：27.5%
- 1995-2006年：12.6%（因收购企业而放缓）

**每股经营收益**（非保险业务）：
- 收益增长加速
- 反映收购策略的转变
- 更多资金用于收购经营企业

## 投资理念

### 规模的双刃剑

承认挑战：
- 大组织容易迟缓、自满
- 但伯克希尔避免了这些陷阱

解决方案：
- 让优秀管理者自主经营
- 避免微观管理
- 保持文化活力

### 长期绩效

关键指标：
- 每股投资增长
- 每股经营收益增长
- 两者平衡发展

### 管理自主权

原则：
- 信任管理者
- 不干预日常经营
- 只在资本配置上决策

## 注释

2006年是伯克希尔创记录的一年，展示了GEICO低成本运营商的力量。巴菲特坦诚承认运气的因素（无重大飓风），但同时也强调了GEICO生产力提升和管理自主权的重要性。这一年也讨论了大企业面临的挑战，以及如何通过分权管理避免规模带来的陷阱。
