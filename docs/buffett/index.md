# 巴菲特致股东信

沃伦·巴菲特致伯克希尔·哈撒韦股东的信，涵盖 1977 年至 2025 年，共 48 封信。

## 📅 时间线

<div class="timeline" markdown>

<div class="timeline-item" markdown>
**1977-1987：奠基时期**

建立投资理念基础，提出 ROE、浮存金、经济商誉等核心概念。
</div>

<div class="timeline-item" markdown>
**1988-1997：理念深化**

市场先生理论、透视盈余、内在价值评估方法的系统阐述。
</div>

<div class="timeline-item" markdown>
**1998-2007：规模扩张**

重大收购时期，BNSF、Marmon、Iscar 等标志性收购。
</div>

<div class="timeline-item" markdown>
**2008-2017：金融危机与转型**

金融危机应对、错误反思、账面价值告别。
</div>

<div class="timeline-item" markdown>
**2018-2025：新时代**

Greg Abel 继任、文化与价值观系统阐述、新收购策略。
</div>

</div>
