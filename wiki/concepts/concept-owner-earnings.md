# 所有者收益

---

## 概述

所有者收益（Owner Earnings）是巴菲特用来评估企业真实盈利能力的关键指标。它代表了企业扣除维持性资本支出后，可以自由分配给股东的现金。巴菲特认为这是评估企业价值的最佳方式。

**类型**：指标  
**类别**：财务分析  
**领域**：价值投资

---

## 核心公式

### 基本定义

```
所有者收益 = 净利润 + 折旧/摊销 - 维持性资本支出 ± 营运资本变化
```

### 简化公式

```
所有者收益 = 净利润 + 折旧 - 资本支出
```

### 与 GAAP 收益的区别

- **GAAP 收益**：按照会计准则计算的收益
- **所有者收益**：企业真正创造的可分配现金

---

## 巴菲特的定义

> "从所有者的角度思考，这意味着：报告收益加上折旧、摊销和其他非现金费用，减去为维持企业长期竞争地位和单位产量所需的资本支出。"

### 关键要点

1. **加回非现金费用**：折旧和摊销不影响现金流
2. **减去维护支出**：不是扩张性投资，而是维持现有地位
3. **判断资本支出的性质**：扩张 vs 维护

---

## 为什么重要？

### 1. 反映真实价值创造

企业是否真正在创造价值？
- 赚取账面利润 ≠ 创造现金流
- 所有者收益 > 账面利润 = 优质企业

### 2. 指导投资决策

- 高所有者收益 → 潜在的高股息
- 高所有者收益 → 偿还债务能力
- 高所有者收益 → 内在价值增长

### 3. 识别经济商誉

真正拥有经济商誉的企业：
- 所有者收益 > 账面利润
- 资本支出主要用于维护
- 竞争优势持久

---

## 应用场景

### 估值

```
内在价值 ≈ 所有者收益之和（折现）
```

### 股息支付

```
股息支付率 = 股息 / 所有者收益
```

### 再投资效率

```
资本回报率 = 所有者收益 / 投入资本
```
