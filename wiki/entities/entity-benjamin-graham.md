# 本杰明·格雷厄姆

---

## 概述

本杰明·格雷厄姆（Benjamin Graham，1894年5月8日 - 1976年9月21日）是一位美国投资者和经济学家，被誉为"价值投资之父"。他系统性地发展了价值投资理念，对现代金融分析产生了深远影响。沃伦·巴菲特称他为"对他投资生涯影响最大的人"。

**类型**：人物  
**角色**：投资大师 / 学者  
**主要贡献**：价值投资理论体系

## 生平简介

### 早年经历

格雷厄姆出生于伦敦的一个移民家庭，后随家人移居纽约。他从哥伦比亚大学获得奖学金，并在那里完成了学业。

### 职业生涯

- 在华尔街从底层做起，最终成为著名的证券分析师
- 创立 Graham-Newman Partnership，这是最早的对冲基金之一
- 在哥伦比亚大学商学院任教，培养了包括巴菲特在内的众多学生

---

## 核心思想

### 1. 内在价值

格雷厄姆认为每项投资都有"内在价值"——基于公司资产、收益和股息确定的真实价值。市场价格会围绕内在价值波动，但长期来看会回归价值。

### 2. 安全边际

> "安全边际总是取决于所支付的价格。"

安全边际是证券内在价值与市场价格之间的差额。即使分析有误，更大的边际也能提供保护。

### 3. 市场先生

格雷厄姆创造了"市场先生"的比喻：
- 市场先生每天提出买卖报价
- 他的情绪每天波动，时而乐观时而悲观
- 聪明的投资者利用市场先生的情绪，而非被其左右

### 4. 控股权益投资

对于普通股民，格雷厄姆建议采用"美元成本平均法"——定期定额投资，避免试图预测市场时机。

## 代表著作

### 《证券分析》(Security Analysis, 1934)

与戴维·多德合著，被誉为"投资者的圣经"。这是第一本系统阐述基本面分析的著作，至今仍被奉为经典。

### 《聪明的投资者》(The Intelligent Investor, 1949)

面向普通投资者的价值投资指南。巴菲特称这是"有史以来最伟大的投资书籍"。

---

## 对巴菲特的影响

沃伦·巴菲特在哥伦比亚大学师从格雷厄姆，并在毕业后加入格雷厄姆的投资合伙企业。格雷厄姆的投资原则成为巴菲特投资哲学的基础：

1. **买股票就是买企业** - 关注企业价值而非股价
2. **安全边际** - 永远不要支付过高的价格
3. **市场先生** - 利用市场非理性而非被其控制
4. **能力圈** - 只投资你理解的企业

巴菲特曾说：
> "格雷厄姆的方法，我皈依了，此后从未动摇。"

---

## 投资原则总结

| 原则 | 说明 |
|------|------|
| **内在价值** | 关注企业真实经济价值 |
| **安全边际** | 购买价格显著低于内在价值 |
| **市场先生** | 利用市场波动而非被其控制 |
| **长期视角** | 不试图预测短期市场 |
| **分散投资** | 至少10-30只股票 |
| **美元成本平均法** | 定期定额投资 |

## 备注

格雷厄姆的理论为整个投资行业奠定了基础。尽管市场在过去几十年发生了巨大变化，但他的核心原则——关注企业价值、保持安全边际、利用市场非理性——仍然被全球投资者遵循。
